# CFPurge — charte graphique

## Idée

CFPurge transforme un flux réseau encombré en un état propre. Le symbole associe deux canaux qui convergent vers un centre vide : le cyan et le bleu racontent le trafic, le point orange signale l’action de purge.

## Palette

| Token | Hex | Usage |
| --- | --- | --- |
| Graphite | `#151922` | Logo, zones fortes, contraste |
| Surface | `#202633` | Cartes et surfaces sombres |
| Cyan signal | `#66E3D0` | Action principale, état sain |
| Bleu réseau | `#4F7CFF` | Information, second canal |
| Orange purge | `#FF9D4D` | Action destructive ou en cours |

## Usage

- `cfpurge-mark.svg` est le signe seul pour l’icône d’application et les petits formats.
- `cfpurge-logo.svg` est la version horizontale pour documentation et présentation.
- `favicon.svg` est optimisé pour les tailles 16–64 px.
- Garder une marge libre d’au moins 12 % autour du signe et ne pas le poser sur un fond orange.
